# chapter 1

## 机器学习善长：
- 传统方法解决需要设计大量规则的问题
- 传统方法无法解决的复杂问题
- 需要不断适应新数据的问题
- 发现复杂问题、大规模数据的内在规律

## 分类
- 监督学习 / 无监督学习 / 半监督学习 / 强化学习
- 批量学习 / 在线学习
- 实例学习 / 模型学习

### 监督学习
- k-NN
- LR
- SVM
- DT、RF
- NN

### 非监督学习
- 聚类：k-means、HCA、EM
- 可视化、降维：PCA、KPCA、LLE、t-SNE
- 关联性规则学习：Apriori、Eclat

## 机器学习的难点

### 数据：
- 训练数据不充分
- 训练数据不具代表性
- 数据质量差
- 不相干的特征

### 算法：
- 过拟合
- 欠拟合
